# POS Backend API

## Technologies
- Django
- Django REST Framework
- MySQL

## Features
- Authentication
- Inventory
- Sales
- M-Pesa Integration
- Reports

## Installation

```bash
git clone https://github.com/ZERODAY-254/POS-BACKEND.git
cd POS-BACKEND
pip install -r requirements.txt
python manage.py migrate
python manage.py runserver
```

## API URL

http://127.0.0.1:8000/api/
